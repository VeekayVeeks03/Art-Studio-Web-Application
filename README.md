# Art Studio Web Application

Art Studio Web Application is a web-based art studio and e-commerce platform built using C# and ASP.NET Web Forms.

The system allows users to browse artworks, register and log in, manage their profiles, view product details, add products to a shopping cart, and complete a checkout or invoice flow. It also includes admin functionality for managing artwork and product listings.

## Project Type

This was a group academic web development project completed by a team of four students.

## Project Status

Completed project.

## Purpose

The purpose of this project was to develop a functional web application that supports online artwork browsing and basic e-commerce features for an art studio environment.

## Main Features

- User registration and login
- Customer profile management
- Artwork and product listings
- Product details page
- Shopping cart functionality
- Checkout and invoice flow
- Admin add, edit, and delete product pages
- Web-based navigation and user interface
